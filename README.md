
# Salary Analysis

## Installation

Clone Project

```bash
  git clone https://github.com/attmhd/salary_analysis.git
```

Move to Project

```bash
  cd salary_analysis/
``` 
Install all libraries needed

```bash
  pip install -r requirements.txt
``` 
Run Streamlit

```bash
  streamlit run main.py
```
